# BABL

Steps to Run 
1. Install CocoaPods
    1.1 Open Terminal
    1.2 Run Command $ sudo gem install cocoapods
    1.3 Goto Project folder
    1.4. Run Command $ pod install
2. Open BABL.xcworkspace
3. Select iphone simulator
4. Run the applecation ( Cmd + R )


## Requirements

- iOS 15.0+ 
- Xcode 13.1
- Swift 5.0
